# 📊 Smart Retail Analytics System

A comprehensive Data Analytics project developed using **Python**, **Power BI**, and **MySQL** to analyze retail sales data, identify customer purchasing patterns, segment customers using RFM analysis, forecast future sales, and create interactive business dashboards for decision-making.

---

# 📌 Project Overview

The Smart Retail Analytics System helps retailers analyze historical sales data and generate valuable business insights through data analytics and visualization. The project integrates data cleaning, customer segmentation, market basket analysis, sales forecasting, and interactive Power BI dashboards to support data-driven business decisions.

---

# 🎯 Objectives

- Clean and preprocess retail sales data.
- Analyze sales performance across categories and regions.
- Perform customer segmentation using RFM Analysis and K-Means Clustering.
- Discover product associations using Market Basket Analysis.
- Forecast future sales trends using Machine Learning.
- Build interactive Power BI dashboards for business intelligence.

# 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/priyanshi0226/Smart-Retail-Analytics-System.git
```

2. Install Python libraries

```
pip install pandas numpy matplotlib scikit-learn mlxtend
```

3. Run Python scripts in the following order

```
data_cleaning.py

customer_segmentation.py

market_basket_analysis.py

sales_forecasting.py

dashboard_data.py

final_report.py
```

4. Open

```
PowerBI/Smart_Retail_Analytics_System_Final.pbix
```

using **Power BI Desktop**.
